Learning Git

this project uses git as a version control system, along with the GitHub website for hosting the code and multiple other features

if you want a good understanding of git, suggest to read the first 3 chapters of https://git-scm.com/book/en/v2:
* the following sections could be skipped: 1.5, 1.6, 2.6, 2.7, 3.5, 3.6
* the book is a recommended resource by the git community but many other tutorials exist, so feel free to use other 'intro to git/github' videos/websites
